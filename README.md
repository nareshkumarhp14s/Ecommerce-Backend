# Full Stack E-Commerce Mobile Application (Backend)

## Overview

Backend API for a full-stack e-commerce application built with Node.js, Express.js, and MongoDB Atlas.

## Features

* JWT Authentication
* Role-Based Authorization
* Product CRUD Operations
* Order Management
* User Management
* Reviews & Ratings
* Razorpay Payment Integration
* Payment Verification
* Cloudinary Image Upload
* PDF Invoice Generation
* Analytics Dashboard

## Tech Stack

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT
* Razorpay
* Cloudinary
* PDFKit

## Environment Variables

Create a `.env` file:

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET
RAZORPAY_KEY_ID=YOUR_KEY
RAZORPAY_KEY_SECRET=YOUR_SECRET
CLOUD_NAME=YOUR_CLOUDINARY_NAME
CLOUD_API_KEY=YOUR_API_KEY
CLOUD_API_SECRET=YOUR_SECRET
```

## Installation

```bash
npm install
npm start
```
